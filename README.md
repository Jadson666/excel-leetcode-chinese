## 背
### 堆疊和列隊
- 單調堆疊的定義、做法和應用場景。
- 兩個堆疊可以模擬列隊，兩個列隊可以模擬堆疊。
### BST
- `中序遍歷`的遍歷順序剛好就是由小排到大
### 二元樹
- 迭代版的前序、中序、後續都能用一個堆疊實現，用`布林標記法`統一其風格，否則`中序`（額外變數）、`後序`（反轉前序）都需要奇技淫巧。
- `中序`遍歷加上任一遍歷（前序或後序），可以建構唯一樹
### 字串
- 記熟 Longest Common Subsequence (LCS) 的定義，還有它如何用 DP 尋找最大值。
### 圖形
- 有向圖找環，用 `DFS` 或是`拓樸排序`
